# Connectivity Test

## Tujuan
Memastikan semua node dan VM saling terhubung
dalam satu jaringan lokal.

## Skenario
- Ping antar node Proxmox
- Ping dari laptop ke VM

## Langkah Pengujian
1. Ping Node 1 → Node 2
2. Ping Node 2 → Node 3
3. Ping Laptop → VM

## Hasil
- Semua ping berhasil
- Tidak ada packet loss signifikan

## Kesimpulan
Koneksi jaringan lokal berjalan dengan baik
dan siap untuk pengujian lanjutan.
